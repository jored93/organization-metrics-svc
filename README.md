# Organization metrics service

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Module operations

```bash
# create a new module
$ nest g mo name

# create a new controller
$ nest g co name/controllers/name --flat

# create a new service
$ nest g s name/services/name --flat
```
